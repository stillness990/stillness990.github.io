使用hexo搭建博客github托管

发表于 2022-06-05

一. 安装hexo

  1. 下载node.js
  1. 解压
  1. 还是移动node.js 文件夹到/usr/local/node文件夹
  1. 编辑~/.bash_profile
  1. 刷新使之生效
  1. 查看版本
  1. 安装cnpm淘宝镜像源
  1. 查看cnpm是否安装好
  1. 接下来使用cnpm安装hexo
  1. 验证是否安装好hexo
  1. 建立一个空的博客文件夹
  1. 初始化blog需要挺长时间
  1. 启动hexo
  1. 新建一个博客文章

hexo clean

16. 生成博客


# 经过hexo处理后就生成了一片博客了

githubID.github.io #githubID为你的帐号名，部署好以后也可以通过这个名字访问你的博客

cnpm install --save hexo-deployer-git

nano _config.yml #vim也可以安装自己的习惯
在# Deployment下新增
type:git #声明git
repo:https://github.com/githubID/githubID.github.io.git #指定仓库名
branch:master #v指定分支
ctrl+o #保存
ctrl+X #退出

hexo d
#部署的过程中会提示输入你的github帐号和密码

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
# 下载的主题在/themes这个文件夹里

nano _config.yml #vim也可以安装自己的习惯
在#Extensions下
theme:yilia
ctrl+o #保存
ctrl+X #退出
hexo clean
hexo g           #重新生成一下
hexo s           #重新启动

hexo d
```
服务器使用nethogs监视网络
