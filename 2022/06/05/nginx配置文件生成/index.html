Nginx 是一个非常高性能的 Web 服务器，具有处理大型站点的能力。由于它能够处理多个并发连接，因此非常受欢迎。由于许多其他原因，它也是首选，例如：
其异步事件驱动架构

内存使用率低

负载均衡

带 SNI 的 TLS/SSL

静态文件的惊人快速处理

带缓存的 FastCGI 支持 (PHP)

反向代理

Linux 用户发现配置此 Web 服务器很容易，但对于新用户来说，这可能会令人困惑。本指南通过演示如何使用NGINXConfig轻松生成这些 Nginx 配置来解决这一难题。
